## Commands
### /coordlist help
### /coordlist view
### /coordlist add <coord name>
### /coordlist add <coord name> [x] [y] [z]
### /coordlist remove <coord name>
### /coordlist track <coord name>
### /coordlist rename <coord name> <new name>
